# my-react
使用ES6 + webpack + react + react-router4 搭建自己的react脚手架

## 目录说明
`src` : 源代码目录

`build`: 打包目录

`src/assets` :存放静态资源

`src/modules` : 模块

`src/style` : 公用样式

## 使用说明
```js
git clone https://github.com/tian-cai/my-react
//安装依赖
npm install
//本地运行
npm run start
//打包
npm run build
```
